Données

Toutes les sources

Chaque source utilisée dans cette analyse est listée ci-dessous avec son statut d'accès, son rôle et un lien direct. Rien n'est caché. Tout est reproductible.

Sources de données

  • Open

    Open-Meteo Historical Weather API

    Données horaires de température à 2m, modèle ERA5 (Copernicus / ECMWF). Coordonnées : 18.0858°N, 15.9785°W. Période : 2019–2025. Licence CC BY 4.0.
    open-meteo.com

  • Open

    OMS / OMM — Définition de nuit tropicale

    Seuil standard : minimum nocturne > 25°C. Utilisé comme référence. Notre analyse utilise 26°C avec justification déclarée.

  • Officiel

    SOMELEC — Communiqués de presse

    Communiqués sur les coupures d'électricité : mars 2026 (panne OMVS pendant Ramadan), août 2025 (maintenance Dar Naim / Teyaret), mai 2020 (perturbations réseau OMVS), avril 2017 (sur-fréquence Manantali). Pas de données quantitatives publiques sur les heures de coupure.

  • Officiel

    AfDB — Desert-to-Power Roadmap Mauritanie (2020)

    Capacité installée SOMELEC (~512–530 MW), mix énergétique (71% thermique), croissance de la demande (6–7% par an), structure d'interconnexion OMVS.
    afdb.org

  • Officiel

    Climatescope 2022 — Mauritanie

    Demande de pointe (~200 MW), capacité disponible (~450 MW), taux d'accès à l'électricité (~55%), tarifs SOMELEC ($156/MWh moyenne).
    global-climatescope.org

  • Déclaré

    Connaissance locale — Coupures d'électricité

    Les coupures d'électricité sont fréquentes à Nouakchott et se concentrent pendant les mois les plus chauds. Déclaré comme contexte, pas comme variable mesurée. Corroboré par les communiqués SOMELEC et la couverture de presse.

  • Déclaré

    Connaissance locale — Exode saisonnier et legzer

    L'exode saisonnier de Nouakchott vers les villages de l'intérieur et les legzer en périphérie est un phénomène connu de tous les habitants. Déclaré comme contexte. Aucune donnée de mobilité saisonnière n'existe pour Nouakchott.

Couverture de presse
